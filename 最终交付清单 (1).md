```bash

```

### 步骤2: 运行脚本
```bash
# 方法A: GUI模式（最简单）
python3 pest_report_extractor.py

# 方法B: 命令行模式（推荐）
python3 pest_report_extractor.py --pdf "你的PDF.pdf" --report --open
```

打开生成的Excel文件，查看两个工作表：
1. 数据分析报告 ⭐
2. 虫害情况

## ✨ v2.0 核心新功能

### 自动生成数据分析报告

添加 `--report` 参数或在GUI中选择"是"，即可生成包含以下内容的报告：

1. **数据概览**
   - 总记录数、虫害总数
   - 平均密度、最大单点

2. **虫害类型统计**
   - 各类型的记录数、总数量、占比

3. **建筑物统计**
   - 各建筑的记录数、总数量、占比

4. **高危区域TOP10**
   - 排名、位置、虫害类型、数量
   - 前三名高亮显示

## ⚠️ 重要提示

### 确保生成报告
运行脚本时必须添加 `--report` 参数：

```bash
# ✅ 正确 - 会生成报告
python3 pest_report_extractor.py --pdf file.pdf --report

# ❌ 错误 - 不会生成报告
python3 pest_report_extractor.py --pdf file.pdf
```

### 查看报告位置
报告在Excel文件的 **第一个工作表**（底部标签："数据分析报告"）
